    How to Use the Quiz Loader
    A quick guide for loading, taking, checking, and retrying your question bank.
    Important: this tool only becomes reliable when your sheet follows the expected columns exactly. If a bank does not load properly, check the headers first before assuming the quiz logic is broken.

    Step 1 Load a valid bank file

    Use Choose bank to pick an XLSX or CSV file, then press Load Bank. The loader reads the first sheet and expects one row per question.

    Required Match the expected columns

    The safest format is ID, Topic, Difficulty, Stem, A, B, C... plus Answer and Explanation. The loader accepts one or more choice columns per question, so one item can have 5 choices while another has 4, as long as the later unused choice cells are just left blank.

    Default Review all questions at once

    The loader now opens in All at once mode by default so you can scan a full set quickly. If you want a tighter drill session, switch to One-by-one.

    Submit Get auto-checking and explanations

    Press Submit when ready. Correct answers are highlighted in green, wrong ones in red, and the explanation block appears if Show explanations is enabled.

    Retry Focus on weak spots

    After submitting, use Retry Wrong to rebuild the pool using only the items you missed or skipped. This is useful for fast, focused review.

    Fallback If XLSX fails, save as CSV

    Some browsers, devices, or offline situations can make spreadsheet parsing flaky. If that happens, export the sheet as CSV and load it again.

    How to Build a Question Bank
    Use a spreadsheet-friendly format, then let the loader turn it into a self-test.
    Disclaimer: the prompts below assume the AI tool you are using can create files directly. If it cannot, ask it to output the bank as a table so you can paste it into Excel, Google Sheets, or another spreadsheet app before saving as XLSX or CSV.

    Format Build one row per question

    Keep the sheet simple: one question per row, one answer per column, and one explanation per row. Avoid merged cells, decorative headers, and multi-row question layouts.

    Sample CSV Format
    ID,Topic,Difficulty,Stem,A,B,C,D,E,Answer,Explanation
Q1,General Anatomy,Easy,Which structure encloses the nucleus?,Cell membrane,Cytoplasm,Nuclear envelope,Golgi apparatus,,C,The nuclear envelope directly surrounds the nucleus, while the cell membrane surrounds the whole cell.
Q2,Biochemistry,Medium,Which molecule is the main immediate energy currency of the cell?,DNA,ATP,RNA,Glycogen,,B,ATP is the immediate usable energy currency, while glycogen is a storage form.

    AI Tip Give the model source files and strict formatting rules

    You will usually get better banks if you upload your source material, tell the AI to use only those files, and ask for the output in the exact spreadsheet columns needed by this loader.
